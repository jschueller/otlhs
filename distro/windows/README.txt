# otlhs module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otlhs\examples\otlhs

Installation
  Silent install can be done with the switch /S (e.g. openturns-otlhs-0.0-py27.exe /S).
  To test otlhs module installation, in an OpenTURNS python console type:
    import otlhs
    print otlhs.__version__

Uninstallation
  OTLHS module will be automatically uninstalled at OpenTURNS removal.
  OTLHS module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
